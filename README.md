# Termux-SSH
Termux SSH helps you to setup SSH server on termux application on android, which helps you to execute tasks remotely.

## How to install

1. Open Termux terminal
2. pkg install git
3. copy the url from the address bar and clone the Termux-SSH
   git clone <url>
4. cd Termux-SSH
5. python install.py
6. python main.py
